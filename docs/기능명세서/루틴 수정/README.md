# 루틴 수정 — 메이트 변경 제한

## 1. 목적

루틴을 생성한 후에는 수정 화면에서 메이트를 제거하거나 다른 메이트로 변경할 수 없도록 한다.

## 2. 기능 정책

- 루틴 수정 시 메이트를 새로 지정할 수 없다.
- 루틴 수정 시 기존 메이트를 다른 사용자로 변경할 수 없다.
- 루틴 수정 시 기존 메이트를 제거할 수 없다.
- 메이트가 지정된 루틴은 수정 화면에 메이트 이름만 표시한다.
- 메이트가 지정되지 않은 루틴은 수정 화면에서 메이트 항목을 표시하지 않는다.

## 3. 제한 이유

향후 벌금 또는 실제 금전과 관련된 기능이 연동될 경우, 루틴 생성 후 메이트 관계를 임의로 변경하면 책임 주체와 벌금 정산 대상이 바뀌는 문제가 발생할 수 있다.

다음과 같은 위험을 예방하기 위해 메이트 수정을 제한한다.

- 벌금이 설정된 루틴에서 메이트를 임의로 제거하는 행위
- 벌금 또는 인증 책임을 피하기 위해 메이트를 임의로 변경하는 행위
- 메이트 변경 전후의 인증 내역, 벌금 책임, 정산 대상이 불일치하는 문제
- 당사자의 동의 없이 금전과 관련된 관계가 변경되는 문제

## 4. 수용 기준

- 수정 화면에 메이트 선택, 변경, 제거를 수행할 수 있는 컨트롤이 없다.
- 메이트가 있는 루틴은 현재 메이트 이름을 읽기 전용으로 표시한다.
- 메이트가 없는 루틴은 메이트 라벨과 입력 영역을 표시하지 않는다.
- 루틴 수정 후에도 기존 메이트 정보가 유지된다.
